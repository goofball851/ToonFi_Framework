# “Built on the Narrative Layer Stack” Badge

This badge certifies **logic compliance**, not visual style.

---

## Badge Meaning

A project displaying this badge confirms:

- Story Boundary respected
- Canon is immutable
- Post-story persistence enforced
- Continuous Motion UI compliant
- No pay-to-authority mechanics

---

## Badge Restrictions

The badge must not be used if:
- users can control narrative outcomes
- canon can be altered
- interaction triggers story events

---

## Recommended Text

> Built on the Narrative Layer Stack  
> Stories end. Existence continues.
