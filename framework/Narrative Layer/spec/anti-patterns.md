# Narrative Layer Stack — Anti-Patterns

These patterns violate the Narrative Layer Stack.

---

## Presentation Traps
- portfolios
- galleries
- carousels
- “view full version”

---

## Control Traps
- play buttons
- start/stop states
- reset animations
- character selectors

---

## Game Traps
- XP systems
- upgrades
- skill trees
- optimization loops

---

## Store Traps
- pay-to-decide
- premium story outcomes
- exclusive canon

---

## Narrative Violations
- sequels disguised as persistence
- infinite canon
- audience-authored plot

If it waits for the user, it is not persistence.
